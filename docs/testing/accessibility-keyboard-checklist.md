# Accessibility Keyboard Checklist

- Skip-to-content link receives focus.
- Header navigation uses links.
- Search dialog opens from the button and with `/`.
- Escape closes dialogs.
- Shop filters use labels and native controls.
- Cart quantity controls are buttons.
- Checkout attestation is a required checkbox.
- Accessibility panel controls persist and apply.
- Focus rings are visible in normal and enhanced focus modes.
